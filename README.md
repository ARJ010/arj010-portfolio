## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ARJ010/your-repo-name.git
```

Navigate to the project:

```bash
cd your-repo-name
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
